## Installation

### Cocoapods(Recommended)

1. Add `pod 'MSSafeContainer'` to your Podfile.
2. Run `pod install`

### Manual

1. Add all files under `MSSafeContainer` to your project

## Requirements

- iOS 5.0 and greater
- ARC/MRC


## How To Use

### Objective-C

```objc

#import "MSSafeContainer.h"
```
